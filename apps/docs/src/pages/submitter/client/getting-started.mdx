# Getting started with the Submitter Client

## Install

### Using a package manager

:::code-group

```bash [npm]
npm install @happy.tech/submitter-client
```

```bash [pnpm]
pnpm add @happy.tech/submitter-client
```

```bash [yarn]
yarn add @happy.tech/submitter-client
```

```bash [bun]
bun add @happy.tech/submitter-client
```

:::
