# Randomness (Alpha)

import { HomePage } from 'vocs/components'

⚠️️⚠️⚠️ **The Randomness service is in Alpha — expect intermittant downtime and potential changes.** ⚠️⚠️⚠️

Randomness is a service that provides a secure and reliable source of randomness for your blockchain applications.
It is designed to be free to consume, easy to use and highly available.

The service utilizes [Drand], combined with a commit and reveal scheme, to provide a synchronous secure source of randomness in every block. 
Read more about the [architecture](/randomness/architecture) or [get started](/randomness/getting-started) with integrating it in your contract.

[Drand]: https://drand.love/docs/
